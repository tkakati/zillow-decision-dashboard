# Roadmap

Near-term roadmap for Zillow Decision Dashboard.

## Current Constraints

- Deterministic ranking and template-style explanatory behavior.
- Frontend-only architecture.

## Next Focus Areas

- Expand tradeoff explanations with stronger contextual reasoning.
- Improve move-in date parsing and input ergonomics.
- Add richer preference dimensions and explainability views.
- Strengthen production-readiness checks and observability.

## To Be Expanded

- Prioritized milestone plan
- Success metrics per milestone
- Rollout and validation strategy
